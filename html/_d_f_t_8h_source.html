9  This is a class that calculates the Discrete Fourier Transform (DFT) of a signal.
10  This is NOT a Fast Fourier Transform, because it is created only to analyse signals for acedamic purposes and not to be used for real time processes.
11  It is a very straight forward aproach represented as a matrix vector multiplication. (Matrix N*K where K == N, multiplied by a vector of size N).
12  Scaling is handled by the algorithm so Inverse Fourier Transorm gives a perfect reconstruction of the original signal if and only if the phases are left unchanged.
13  For use: 1 - create DFT object.
14  2 - call init method.
